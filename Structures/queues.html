What is a queue?

A queue is used for storing and managing data in a specific order.
It follows the principle of “First in, First out” (FIFO), where the first element added to the queue is the first one to be removed.

When would you use a queue?

  • For task scheduling in operating systems
  • Task scheduling in operating systems
  • When you want to get items out in the same order that you put them in, use a queue
