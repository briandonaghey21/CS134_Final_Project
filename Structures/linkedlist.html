What is a linked list?

A linked list consists of nodes where each node contains data and a reference (link) to the next node in the sequence. This allows for dynamic memory allocation and efficient insertion and deletion operations compared to arrays.

When would you use a linked list?

  • When you need to store a variable number of elements that might have different types or sizes.
  • Preferable when you need to perform frequent insertion or deletion operations at any position
  • You do not care about the time it takes to random accessly, sort, or search the list.
